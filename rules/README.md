# AI Coding Rules Entry

This folder is the mandatory rules entry for AI coding tools working on this project.

Before every coding session, every AI coding tool must read and follow:

1. [`../PRD.md`](../PRD.md)
2. [`frontend-ai-coding-rules.md`](frontend-ai-coding-rules.md)

These rules are tool-agnostic and apply to Claude Code, Cursor, Gemini, Copilot, Codex, and any other AI coding tool used in this repository.

If a tool supports project rules, memory, custom instructions, or context files, configure it to always include this folder. Do not rely on chat memory alone.
